My Game Curator: 맞춤형 게임 추천 앱

프로젝트 개요
My Game Curator는 사용자의 게임 성향 테스트 결과와 다양한 필터 기준(난이도, 플레이 시간, 장르 등)을 기반으로 맞춤형 게임을 추천해 주는 큐레이션 애플리케이션입니다.
방대한 게임 데이터 속에서 사용자에게 가장 적합한 '인생 게임'을 찾는 것을 목표로 합니다.

주요 기능 (Features)
1. MBTI 스타일 성향 테스트 기반 추천:
사용자가 간단한 5가지 질문에 응답하면 성향에 따라 점수가 부여됩니다.
2. 이를 기반으로 액션, RPG, 공포, 힐링 등 6가지 장르 중 가장 선호하는 장르 2개를 선정하여 맞춤 필터로 적용합니다.
3. 오늘의 추천 게임: 매일 자정(00시 00분)에 자동으로 초기화되어 평점이 높은 게임 중 무작위로 한 개를 선정하여 메인 화면 상단에 노출합니다. (로컬 데이터 및 SharedPreferences 관리)
4. 다중 필터 및 검색 기능: 키워드 검색, 난이도, 예상 플레이 시간(짧음/중간/김), 멀티플레이 여부 등 상세 필터링 기능을 제공하여 원하는 조건에 맞는 게임을 정확하게 탐색할 수 있습니다.
5. 찜 목록 (Favorites): 사용자가 관심 있는 게임을 찜 목록에 추가하고 관리할 수 있습니다.
6. 게임 상세 정보 모달: 카드 클릭 시 게임의 평점, 난이도, 장르(한국어 변환), 이미지 등의 상세 정보를 모달 형태로 제공합니다.

구분,기술 스택,설명
언어 및 프레임워크,"Kotlin, Jetpack Compose",안드로이드의 최신 언어 및 선언형 UI 프레임워크를 사용하여 효율적이고 반응성 높은 UI를 구현했습니다.
상태 관리,"remember, mutableStateOf, rememberCoroutineScope",Compose 환경에서 로컬 상태 및 비동기 데이터를 효율적으로 관리했습니다.
아키텍처/비동기,"Coroutines, Retrofit","비동기 네트워크 통신 및 데이터 처리를 위해 코루틴을 사용했으며, REST API 통신을 위해 Retrofit 라이브러리를 사용했습니다."
데이터 소스,RAWG API,방대한 게임 정보를 제공하는 RAWG API를 메인 데이터 소스로 활용했습니다.
이미지 처리,Coil,URL 이미지를 효율적으로 로드하고 캐싱하기 위해 사용했습니다.
로컬 저장소,SharedPreferences,"'오늘의 게임' 정보(날짜/ID) 및 '사용자 선호 장르', '찜 목록'과 같은 간단한 데이터를 영구 저장하기 위해 사용했습니다."
